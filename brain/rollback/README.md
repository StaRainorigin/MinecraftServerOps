# `brain/rollback/` · 快照回滚

利用阶段一创建的事务性快照，将整个实例目录无痕迹恢复到崩溃前一刻。

## 文件职责

| 文件 | 职责 |
|------|------|
| `restorer.py` | 绝对事务回滚执行器：读取 `SnapshotRef`，将目录恢复到初始快照状态 |

## 触发场景

1. **死循环熔断**：`MAX_RETRIES = 3` 触顶，立即绝对事务回滚 + 终止自愈 + 呼叫人工。
2. **Critic 不及格**：阶段五打分 `< 60`（如修复致 TPS 掉到 5.0），拒绝成果 + 强制回滚。
3. **RLHF 踩反馈**：用户在 ChatOps 卡片点 👎，立即回滚（见 `memory/chatops/feedback.py`）。

## 设计约束

- **绝对无痕**：回滚必须完整恢复到崩溃前一秒，不留任何修改痕迹。
- **事务性**：回滚过程本身需保证原子性，中途失败不能留下半恢复状态。
